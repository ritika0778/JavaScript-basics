
# JavaScript Basics 🚀

This repository is a collection of projects and mini-games I built while learning **JavaScript** fundamentals. Each project focuses on practicing core concepts like DOM manipulation, event handling, functions, loops, conditionals, and basic logic building — all using **HTML, CSS, and vanilla JavaScript**.

## 📚 About This Repo

This is a personal learning repo, so the code here reflects my growth as a beginner developer. The goal wasn't to build production-ready apps, but to **understand how JavaScript works** by building small, fun, and interactive things.

## 🎮 Projects Included

| Project | Description | Concepts Practiced |
|---------|-------------|---------------------|
| Rock Paper Scissors | Classic game against the computer | Functions, conditionals, random numbers |
| Tic Tac Toe | Two-player browser game | DOM manipulation, arrays, event listeners |

## 🛠️ Tech Stack

- **HTML5** – structure  
- **CSS3** – styling  
- **JavaScript (ES6)** – logic and interactivity

## ▶️ How to Run

1. Clone this repository
   ```bash
   git clone https://github.com/your-username/javascript-basics.git
   ```
2. Open any project folder
3. Open the `index.html` file in your browser

No build tools or dependencies required — just plain JavaScript!
